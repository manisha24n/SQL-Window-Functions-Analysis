# SQL Window Functions

This project demonstrates SQL Window Functions using sample datasets.

## Tables
- daily_revenue
- employees

## Concepts
- Cumulative Revenue
- Department Average Salary
- Employee Salary Ranking
- LAG & LEAD Functions
- Revenue Difference
- Highest Paid Employee per Department

## Functions Used
SUM() OVER(), AVG() OVER(), RANK(), LAG(), LEAD(), ROW_NUMBER()

## Tools
SQL | PostgreSQL | MySQL
